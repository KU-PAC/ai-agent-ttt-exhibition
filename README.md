このレポジトリはSO-100系アームロボットと、Unityで作成されたキャラクターAgentを組み合わせ、人とAIがシンプルな〇✕ゲーム（三目並べ）を行うデモンストレーション用のレポジトリです。

フォルダの構造について、下記の通り定めます。
- agent:3DキャラクターなどUnity関連の実装をするモジュール
- master:全体の制御を行う最上位モジュール
- models:盤や駒のCADデータ
- robot:SO-101コントロール用のモジュール

各フォルダの担当者は担当外のフォルダを改変しないようにしてください。
各フォルダ内に.gitignoreを配置し、不要なファイル・API鍵などの機密情報を除外するようにしてください。
