このレポジトリはKUPACのSusHi Tech Tokyo 2026出展に向けた、SO-100系アームロボットと、Unityで作成されたキャラクターAgentを組み合わせ、人とAIがシンプルな〇✕ゲーム（tic-tak-toe）を行うデモンストレーション用のレポジトリです。

フォルダの内容は下記の通りです。
- agent:3DキャラクターなどUnity関連の実装をするモジュール
- master:全体の制御を行う最上位モジュール
- models:盤や駒のCADデータ
- robot:SO-101コントロール用のモジュール
- vision:盤面認識技術用のモジュール
